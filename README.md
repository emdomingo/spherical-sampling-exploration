# Spherical Sampling Exploration

An exploration of sampling schemes on the sphere and their reconstruction accuracy.

## Notebooks

This repository includes the following notebooks:

| Notebook | Description |
|---|---|
| `00_mathematical_background.ipynb` | A notebook with mostly markdown blocks that show the background of Spherical Sampling and Spherical Harmonics |
| `01_classical_grids.ipynb` | An exploration of the Classical Grids: DH, GL, and MW |
| `02_polyhedral_grids.ipynb` | An exploration of the Polyhedral Grids: Tetra, Octa, and Icosa |
